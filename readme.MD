# Loja - Bird Sound
Esse projeto, desenvolvido como TCC do curso de Programação Web no Instituto da Oportunidade Social, é uma loja virtual inspirada pela Bird Sound, que trabalha com a venda e instalação de equipamentos e acessórios automotivos.  
Dentro do contexto do projeto, o site apresenta as seguintes funcionalidades:
- Exibir informações relevantes sobre a loja (endereço, serviços, formas de contato, horário de funcionamento)
- Descrever os serviços da loja com um botão "Solicitar orçamento" que envia uma mensagem personalizada para o WhatsApp da loja
- Dispor um catálogo de produtos que o usuário pode filtrar,  visualizar, adicionar ao carrinho
- Carrinho de compras dinâmico que dispõe todos os produtos adicionados, com opções de remoção, aumento ou redução de quantidade, cálculo de valor final e opções de parcelamento. Por fim, permitindo ao usuário enviar todas as informações do pedido já organizadas em uma mensagem personalizada para o WhatsApp da loja.
## Linguagens
O projeto usa HTML, CSS e Javascript vanilla. Os produtos são organizados em um arquivo JSON com todas as informações relevantes e buscados para exibição no site.
